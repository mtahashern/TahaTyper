# How to Run TahaTyper Peacefully 🚀

Since TahaTyper is a custom-built app, Windows might show a warning. This is normal for independent software!

### 1. Bypassing Windows SmartScreen
*   Click **"More info"**.
*   Click **"Run anyway"**.

### 2. Laptop-Specific Tips 💻
If the hotkeys (F6/F7) aren't working on your laptop:
*   **The "Fn" Key**: Most laptops require you to hold the **`Fn`** key while pressing F6 or F7. Try **`Fn + F6`** to start and **`Fn + F7`** to stop.
*   **Window Focus**: After clicking START (or pressing F6), you have 3 seconds to click into the window where you want the text to appear. Make sure that window is "active."
*   **Run as Administrator**: If it's still not typing, right-click `TahaTyper.exe` and select **"Run as Administrator."**

### 3. Screen Scaling
If the window looks too big or you can't see the buttons:
*   We've added a **Scrollbar** on the right side! Just scroll down to reach the START and STOP buttons.
*   The UI is now compact and should fit perfectly on 13-inch and 15-inch laptop screens.
